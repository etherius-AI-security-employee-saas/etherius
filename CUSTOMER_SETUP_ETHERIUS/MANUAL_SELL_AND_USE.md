# Etherius Delivery Manual Index

Use the new separate manuals in this folder:

1. `CUSTOMER_README.md`  
   Quick overview of what each person uses.
2. `CUSTOMER_ADMIN_MANUAL.md`  
   Full customer admin steps.
3. `EMPLOYEE_MANUAL.md`  
   Full employee install and enrollment steps.

## What to send customer

Send entire `CUSTOMER_SETUP_ETHERIUS` folder.

Customer gets:

1. `admin\` executables and startup bat
2. `employee\` executables and startup bat
3. the 3 manuals above
4. subscription key (from provider/CEO)

Do not send:

1. provider CEO master key
2. provider private backend credentials
3. source code folders
