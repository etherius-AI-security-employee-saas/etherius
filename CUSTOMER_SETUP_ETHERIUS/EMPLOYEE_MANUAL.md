# Employee Manual (Simple)

Use this if you are an employee installing Etherius Shield.

## 1) What you need from your admin

Before installation, ask your admin for:

1. Company Enrollment Code
2. Employee Key
3. Backend URL (if required)

## 2) Start shield

1. Open `employee` folder.
2. Double-click `EMPLOYEE_START_SHIELD.bat`
   - or run `EtheriusShield.exe`

## 3) Enroll your device

In the app, fill:

1. Company Enrollment Code
2. Employee License Key
3. Backend URL (if not prefilled)

Then click **Enroll This Device**.

## 4) Start protection

1. Click **Start Protection** (if not already started automatically).
2. Confirm status shows protection active/connected.

## 5) Email and suspicious activity check

- You can paste suspicious email content in the shield app for analysis.
- The app sends telemetry to the security dashboard.

## 6) If something fails

If enrollment fails:

1. Re-check enrollment code and employee key
2. Check internet/network connection
3. Confirm backend URL from admin is correct
4. Contact your admin for new key if old key expired/revoked

If protection is not active:

1. Reopen shield app
2. Re-enroll if needed
3. Click **Start Protection**

## 7) Privacy and responsibility

- Do not share your employee key with others.
- Do not disable shield without admin approval.
