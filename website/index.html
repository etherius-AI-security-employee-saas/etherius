Frequently Asked Questions

Is the agent visible to employees?

The agent runs silently in the background. Employees see a system tray icon but cannot disable it without admin rights.

Does it work on personal devices?

Etherius is designed for company-owned devices and requires admin installation plus an agent token from your dashboard.

How is my data protected?

All data is encrypted in transit (TLS 1.3) and at rest. Each company is isolated with strict multi-tenant boundaries.

Can I try it for free?

Yes. The Starter plan is free for up to 5 endpoints and no credit card is required.

What happens if the agent loses internet connection?

The agent stores events locally and syncs when the connection is restored. No data is lost.
