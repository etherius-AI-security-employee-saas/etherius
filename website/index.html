Activation Flow

1

CEO issues customer subscription key with fixed employee quantity.

2

Customer manager installs same software and activates with subscription key.

3

Manager generates employee keys inside software (within purchased seat limit).

4

Employees activate in same software and protection data streams to manager dashboard.

CEO Manual

  1. Open CEO console and set API URL + CEO master key.
  2. Issue subscription key with employee quantity and validity days.
  3. Share subscription key privately with customer manager.
  4. Monitor all companies, seats used, online endpoints, alerts, and login/logout counts.

Customer Manager Manual

  1. Install Etherius setup and open software.
  2. Switch to Manager View and activate company with subscription key.
  3. Sign in to unlock in-app dashboard and generate employee activation keys.
  4. Track endpoints, alerts, protection status, and employee login/logout activity.

Employee Manual

  1. Install the same Etherius setup.
  2. Stay in Employee View and enter company code + employee key.
  3. Enroll device, apply activation code, start protection.
  4. Run AI quick/deep scans while manager receives threat telemetry.

Cloud Requirements

  • Public website domain for distribution and manuals.
  • Secure API domain for activation, telemetry, and manager dashboard data.
  • Production database (PostgreSQL recommended) for persistent customer and endpoint data.
