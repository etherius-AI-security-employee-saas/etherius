Frequently Asked Questions

Is the agent visible to employees?

The agent runs silently in the background. Employees see a system tray icon but cannot disable it without admin rights.

Does it work on personal devices?

Etherius is designed for company-owned devices and requires admin installation plus a valid agent token.

How is my data protected?

All data is encrypted in transit (TLS 1.3) and at rest. Company data remains fully isolated by tenant.

Can I try it for free?

Yes. The Starter plan is free for up to 5 endpoints with no credit card required.

What if the agent loses internet?

The agent stores events locally and syncs automatically when connectivity is restored.
