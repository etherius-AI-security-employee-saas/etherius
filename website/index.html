Activation and Access Flow

1

CEO issues a customer subscription key with fixed employee seat quantity.

2

Customer manager installs the same software and activates manager access using that key.

3

Manager creates employee activation keys inside manager dashboard, limited by purchased seats.

4

Employees activate from employee section and endpoint telemetry appears in manager dashboard.

Manager View

Available only for activated customer managers. Includes employee status, alerts, policies, and reports.

Employee View

Focused on device enrollment, runtime protection, and scans. No manager admin controls are exposed.

Single Complete Tutorial

  1. CEO creates subscription key in CEO control console with seat count (example: 300).
  2. Customer manager installs Etherius setup and activates using that subscription key.
  3. Manager signs in, generates employee keys, and shares each key to employees.
  4. Employee installs same setup, activates employee mode, and starts protection.
  5. Manager monitor panel inside software shows endpoint state, alerts, and login/logout activity.
Is manager dashboard public on this website?

No. This page only distributes installer and tutorial information. Manager dashboard access stays inside software and requires valid subscription plus manager login.

Do employees see manager controls?

No. Employee role has protection and scan functions only. Policy and administrative controls remain restricted.

Do I need separate setup for employee and manager?

No. One installer package is used for both roles; access is determined by activation and sign-in permissions.
