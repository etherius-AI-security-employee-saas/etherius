# Etherius Start Here

## Live Links

1. Website: `https://etherius-security-site.vercel.app`
2. API: `https://etherius-security-api.vercel.app`

## One-Setup Rule

Install one software only: `EtheriusSuite.exe`

1. Managers and employees use the same setup.
2. Role access is controlled by activation key and login.
3. No public web dashboard is required for customer operations.

## CEO Quick Steps

1. Run `ceo/CEO_START_CONSOLE.bat`.
2. Issue subscription key with employee quantity (example: 300).
3. Send key privately to customer manager.

## Customer Manager Quick Steps

1. Open software and switch to `Manager View`.
2. Activate company with subscription key.
3. Sign in to unlock manager dashboard.
4. Generate employee keys.
5. Monitor endpoint status, alerts, and login/logout metrics.

## Employee Quick Steps

1. Open software in employee mode.
2. Enter company code + employee key.
3. Enroll and start protection.
4. Run AI scans if needed.

## Security Notes

1. No subscription key means no manager activation.
2. Expired subscription blocks manager and employee API actions.
3. Employee mode does not provide manager controls.
4. CEO endpoints require `X-CEO-Key`.
